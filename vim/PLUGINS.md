# Vim Plugin List

## Shared (.vimrc / .ideavimrc で揃えるプラグイン)

プラグインを追加・削除するときは両方のファイルを更新すること。
コード内では `" [shared]` コメントで識別できる。

| プラグイン | 機能 |
|-----------|------|
| tpope/vim-commentary | `gc` でコメントアウト |
| tpope/vim-surround | `ys`/`cs`/`ds` で括弧・クォート操作 |
| tpope/vim-unimpaired | `[`/`]` 対称キーマップ集 |
| machakann/vim-highlightedyank | ヤンク範囲をハイライト |

## .vimrc のみ

| プラグイン | 機能 |
|-----------|------|
| morhetz/gruvbox | カラースキーム |
| nelstrom/vim-visual-star-search | ビジュアル選択で `*` 検索 |
